My Work on RoboRun

Game Summary

The game runs on the unreal engine and the code is written in c++. I worked in a group of 10 developers and worked with the design team and engineering team. My two roles were engineer and sound designer.

Worked On

I worked on a Power up system which gives the players the ability to pick up a power by walking up to it. The different powers that I worked on were invisibility and increased movement speed. To make the player invisible the meshrenderer is turned off when a player activates the ability. To activate the power up the player has to press LMB(left mouse button).

Goal

The goal was to create a high performing pickup that can easily be expanded to include other temporary powers (super-jump or bullet time). To achieve this, i opted to use c++ to write the code instead of blueprints which is UnrealÂ´s visual scripting system.

Challenges

The base structure of the pickup was implemented without much trouble, the player could approach the pickup object and picked it up upon contact. I had the most trouble when i was trying to figure out how to make the player invisible. After copious amount of reading forums i managed to make the character invisible by turning off the meshrenderer. That resulted in Unreal crashing. This problem was solved by having a null check in the c++ code.
